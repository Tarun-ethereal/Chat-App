<div style="max-width: 800px; margin: 0 auto; font-family: Arial, sans-serif; line-height: 1.6;">
  <h1 style="text-align: center; color: #333;">Chat-App</h1>
  
  <p style="color: #555; font-size: 18px;">
    <strong>Chat-App</strong> is a real-time messaging application designed to provide seamless communication between users. Built with React, Node.js, and Tailwind CSS, it delivers instant message updates, a modern responsive interface, and secure user authentication for a smooth chatting experience.
  </p>

  <h2 style="color: #2c3e50; border-bottom: 2px solid #ddd; padding-bottom: 10px;">Features</h2>
  <ul style="list-style-type: disc; padding-left: 20px; color: #555; font-size: 16px;">
    <li><strong>Real-Time Messaging:</strong> Send and receive messages instantly using Socket.IO for live updates.</li>
    <li><strong>User Authentication:</strong> Secure login and registration with proper session handling and cookies.</li>
    <li><strong>Responsive Design:</strong> Tailwind CSS ensures the interface adapts to mobile, tablet, and desktop screens.</li>
    <li><strong>Modern UI:</strong> Clean and intuitive interface built using React components.</li>
    <li><strong>Client-Server Communication:</strong> Axios is used for API calls to the backend, ensuring smooth data exchange.</li>
  </ul>

  <h2 style="color: #2c3e50; border-bottom: 2px solid #ddd; padding-bottom: 10px;">Tech Stack</h2>

  <h3 style="color: #34495e; font-size: 20px;">Frontend:</h3>
  <ul style="list-style-type: disc; padding-left: 20px; color: #555; font-size: 16px;">
    <li>ReactJS for UI development</li>
    <li>Tailwind CSS for responsive styling</li>
    <li>React Router DOM for client-side routing</li>
    <li>Axios for API requests</li>
    <li>Socket.IO Client for real-time messaging</li>
  </ul>

  <h3 style="color: #34495e; font-size: 20px;">Backend:</h3>
  <ul style="list-style-type: disc; padding-left: 20px; color: #555; font-size: 16px;">
    <li>Node.js for server-side JavaScript runtime</li>
    <li>Express.js for building API routes</li>
    <li>Socket.IO Server for handling real-time events</li>
    <li>Js-cookie for managing session tokens</li>
  </ul>

  <h2 style="color: #2c3e50; border-bottom: 2px solid #ddd; padding-bottom: 10px;">Installation &amp; Setup</h2>
  <ol style="padding-left: 20px; color: #555; font-size: 16px;">
    <li>Clone the repository:
      <pre style="background-color: #f4f4f4; padding: 6px; border-radius: 4px;"><code>git clone https://github.com/Tarun-ethereal/Chat-App.git
cd Chat-App</code></pre>
    </li>
    <li>Navigate to the client directory:
      <pre style="background-color: #f4f4f4; padding: 6px; border-radius: 4px;"><code>cd client</code></pre>
    </li>
    <li>Install client-side dependencies:
      <pre style="background-color: #f4f4f4; padding: 6px; border-radius: 4px;"><code>npm install</code></pre>
    </li>
    <li>Start the development server:
      <pre style="background-color: #f4f4f4; padding: 6px; border-radius: 4px;"><code>npm run dev</code></pre>
      <p>Server runs typically on <code>http://localhost:5173</code>.</p>
    </li>
    <li>Setup backend (if included):
      <p>Install server dependencies and start the server in the project root.</p>
    </li>
  </ol>

  <h2 style="color: #2c3e50; border-bottom: 2px solid #ddd; padding-bottom: 10px;">Usage</h2>
  <p style="color: #555; font-size: 16px;">Users can register or log in to the app, join chat sessions, and send messages in real-time. Online presence and message delivery happen instantly through Socket.IO.</p>
